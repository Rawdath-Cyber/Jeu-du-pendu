# Jeu du pendu

## Description

Ce projet est un jeu de devinette de mot développé en Python.

Le joueur doit découvrir un mot secret. Il peut soit proposer une lettre, soit tenter de deviner directement le mot complet.

Chaque erreur coûte une vie. Le joueur dispose de cinq vies au départ pour trouver la bonne réponse.

## Fonctionnalités

- Mot secret prédéfini.
- Système de vies limité à cinq tentatives.
- Possibilité de proposer une lettre.
- Possibilité de proposer directement le mot complet.
- Détection des lettres présentes dans le mot.
- Affichage du nombre de vies restantes.
- Révélation du mot secret en cas de défaite.

## Technologies utilisées

- Python 3

## Concepts utilisés

- Boucles `while`
- Conditions `if`, `elif` et `else`
- Variables
- Chaînes de caractères
- Entrées utilisateur avec `input()`
- Gestion d'un système de vies

## Objectif pédagogique

Ce projet a été réalisé dans le cadre de mon apprentissage de Python afin de pratiquer les structures conditionnelles, les boucles et la manipulation des chaînes de caractères.
